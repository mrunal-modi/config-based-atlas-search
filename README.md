## Overview

This project allowing users to deploy MongoDB Atlas Search by updating a simple config file.

## Pre-requisites
- [Next.js](https://nextjs.org/)
- [Auth0](https://manage.auth0.com/)
- [MongoDB Atlas Search](https://cloud.mongodb.com/)
